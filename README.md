# 網頁切版 - ART NFT

網頁切版練習，包含響應式 RWD

## 使用技術

* Gulp 任務管理
* EJS Layout 版型管理
* Bootstrap 5 變數與元件客制化
* Swiper 輪播效果
* Masonry 瀑布流
